# ❤️ Heart Attack Dashboard

## Overview

This project presents an interactive dashboard built to analyze heart attack-related data and uncover key insights into risk factors and patient conditions.

The dashboard aims to help users better understand patterns, correlations, and indicators that may contribute to heart attack occurrences through clear and intuitive data visualization.

---

## Objectives

* Analyze heart attack dataset to identify key risk factors
* Visualize important metrics in an easy-to-understand dashboard
* Support data-driven insights for health-related decision making

---

## Dataset

The dataset used in this project contains medical and demographic attributes such as:

* Age
* Sex
* Chest pain type
* Blood pressure
* Cholesterol levels
* Fasting blood sugar
* Resting ECG results
* Maximum heart rate
* Exercise-induced angina
* ST depression (oldpeak)
* Number of major vessels
* Thalassemia
* Target (presence of heart disease)

---

## Dashboard Features

* 📌 **Key Metrics Overview**: Summary of important indicators
* 📊 **Data Visualization**: Charts showing relationships between variables
* 🔍 **Interactive Filtering**: Explore data based on specific conditions
* 📉 **Trend Analysis**: Identify patterns related to heart attack risk

## Key Insights

* Age and cholesterol levels show correlation with heart attack risk
* Certain chest pain types are more associated with positive cases
* Higher resting blood pressure may indicate increased risk
* Exercise-induced angina is a strong indicator

---

## Conclusion

This dashboard provides a simple yet powerful way to explore heart attack data and identify important health patterns. It can be further developed into predictive modeling or integrated with real-time health systems.
